# draw-steel
This is simply a repository of content that I have created for the Draw Steel game system that is published by MCDM Productions LLC.

Much of the content is actually in the form of Forge Steel sources books (see: https://github.com/andyaiken/forgesteel) that I have created to run my games.  Some of which includes additons to my Forge Steel library that comes from adventures, or adaptions of them,;[p] published by MCDM Productions and other creators.

This repo began with the intention that others need not repeat the work that I have done with data entry, and it's easier to maintain a git repo than a website.

If your content is misrepresented or otherwise not treated as you would like, please let me know and I will revise or remove it.
